# 成果物概要
親クラス hito のメンバ変数(private)として、レベル：m_level (int 型）と現在
の歩数：m_step (int 型）をもっている。move()メンバ関数（public) が呼ばれると、現在
の歩数が+1 され、現在の歩数が 2，4，8，16，32，，，，のように，2 のべき乗のタイミング
でレベルがあがる。
子クラス runner のメンバ変数(private):として、疲労度あい：m_tired（int 型）をもってい
る。move()メンバ関数は、現在の歩数が＋3 される、レベルは hito と同じように上がる。
歩数 20 歩ごとに、1 回休憩が必要（move しても動けない。1 回休憩したら疲労度合いを 0
に戻す）
